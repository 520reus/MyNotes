\doxysubsection{详细描述}
trajectory rollout class for a non-\/holonomic robot with differential drive constraints 

The \mbox{\hyperlink{classlattice__planner_1_1TrajectoryRollout}{Trajectory\+Rollout}} class provides methods to calculate the next position and velocity from an initial position and velocity, according to the robot\textquotesingle{}s acceleration and velocity limits.
