\doxysubsection{详细描述}
Edge defining the cost function for keeping a minimum distance from inflated obstacles. 

The edge depends on a single vertex $ \mathbf{s}_i $ and minimizes\+: ~\newline
 $ \min \textrm{penaltyBelow}( dist2point, min_obstacle_dist ) \cdot weight_inflation $. ~\newline
Additional, a second penalty is provided with ~\newline
 $ \min \textrm{penaltyBelow}( dist2point, inflation_dist ) \cdot weight_inflation $. It is assumed that inflation\+\_\+dist $>$ min\+\_\+obstacle\+\_\+dist and weight\+\_\+inflation $<$$<$ weight\+\_\+inflation. {\itshape dist2point} denotes the minimum distance to the point obstacle. ~\newline
{\itshape penalty\+Below} denotes the penalty function, see penalty\+Bound\+From\+Below() ~\newline
