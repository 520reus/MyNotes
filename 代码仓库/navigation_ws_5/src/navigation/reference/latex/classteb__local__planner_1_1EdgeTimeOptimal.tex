\doxysubsection{详细描述}
Edge defining the cost function for minimizing transition time of the trajectory. 

The edge depends on a single vertex $ \Delta T_i $ and minimizes\+: ~\newline
 $ \min \Delta T_i^2 \cdot scale \cdot weight $. ~\newline
{\itshape scale} is determined using the penalty\+Equality() function, since we experiences good convergence speeds with it. ~\newline
{\itshape weight} can be set using set\+Information() (something around 1.\+0 seems to be fine). ~\newline
