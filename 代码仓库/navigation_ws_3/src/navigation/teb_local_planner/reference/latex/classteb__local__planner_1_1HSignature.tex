\doxysubsection{详细描述}
The H-\/signature defines an equivalence relation based on homology in terms of complex calculus. 

The H-\/\+Signature depends on the obstacle configuration and can be utilized to check whether two trajectores belong to the same homology class. Refer to\+: ~\newline

\begin{DoxyItemize}
\item S. Bhattacharya et al.\+: Search-\/based Path Planning with Homotopy Class Constraints, AAAI, 2010 
\end{DoxyItemize}

Calculate the H-\/\+Signature of a path 

The implemented function accepts generic path descriptions that are restricted to the following structure\+: ~\newline
The path is composed of points T and is represented by a std\+::vector$<$ T $>$ or similar type (std\+::list, std\+::deque, ...). ~\newline
