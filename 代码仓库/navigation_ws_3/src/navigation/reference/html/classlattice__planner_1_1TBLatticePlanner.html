详细描述

planner core.

This class provides the link from move_base to the lattice planner. The class is written as a plugin for the ros navigation stack and inherits from the nav_core::BaseGlobalPlanner class

参见
http://wiki.ros.org/navigation/Tutorials/Writing%20A%20Global%20Path%20Planner%20As%20Plugin%20in%20ROS for more information about how to write a global planner plugin
