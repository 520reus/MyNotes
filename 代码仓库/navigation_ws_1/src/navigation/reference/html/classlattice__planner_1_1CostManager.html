详细描述

Class for cost calculation for path search

Calculates the costs of a path (or path segment) according to a given cost function. The cost function is a weighted sum of different cost aspects:

  • execution time: cost according to the time it takes to follow the plan.
  • path length: costs according to the traveled distance along a path.
  • environmental costs: costs defined in the dynamic environment, obtained from a layered dynamic cost map.
  • necessary turns: costs according to the rotation the robot has to execute to follow the path.
