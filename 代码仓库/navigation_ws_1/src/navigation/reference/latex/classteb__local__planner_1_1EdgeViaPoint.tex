\doxysubsection{详细描述}
Edge defining the cost function for pushing a configuration towards a via point 

The edge depends on a single vertex $ \mathbf{s}_i $ and minimizes\+: ~\newline
 $ \min dist2point \cdot weight $. ~\newline
{\itshape dist2point} denotes the distance to the via point. ~\newline
{\itshape weight} can be set using set\+Information(). ~\newline
