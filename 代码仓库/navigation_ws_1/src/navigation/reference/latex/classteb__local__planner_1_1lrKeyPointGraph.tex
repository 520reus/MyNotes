Create a graph containing points in the global frame that can be used to explore new possible paths between start and goal. 

This version of the graph creation places a keypoint on the left and right side of each obstacle w.\+r.\+t to the goal heading. ~\newline
All feasible paths between start and goal point are extracted using a Depth First Search afterwards. ~\newline
This version works very well for small point obstacles. For more complex obstacles call the create\+Prob\+Roadmap\+Graph() method that samples keypoints in a predefined area and hopefully finds all relevant alternative paths.
