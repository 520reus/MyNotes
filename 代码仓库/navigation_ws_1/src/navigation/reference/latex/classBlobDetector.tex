\doxysubsection{详细描述}
Detect blobs in image (specialized for dynamic obstacles in the costmap) 

This class is based on Open\+CV\textquotesingle{}s blob detector cv\+::\+Simple\+Blob\+Detector. It has been modified and specialized for dynamic obstacle tracking in the costmap\+: -\/$>$ The modified version also returns contours of the blob.

See \href{http://docs.opencv.org/trunk/d0/d7a/classcv_1_1SimpleBlobDetector.html}{\texttt{ http\+://docs.\+opencv.\+org/trunk/d0/d7a/classcv\+\_\+1\+\_\+1\+Simple\+Blob\+Detector.\+html}} for the original class.
