\doxysubsection{详细描述}
Edge defining the cost function for penalzing a specified turning direction, in particular left resp. right turns 

The edge depends on two consecutive vertices $ \mathbf{s}_i, \mathbf{s}_{i+1} $ and penalizes a given rotation direction based on the {\itshape weight} and {\itshape dir} ( $ dir \in \{-1,1\} $) {\itshape dir} should be +1 to prefer left rotations and -\/1 to prefer right rotations ~\newline
{\itshape weight} can be set using set\+Information(). ~\newline
