\doxysubsection{详细描述}
Edge defining the cost function for limiting the translational and rotational acceleration. 

The edge depends on five vertices $ \mathbf{s}_i, \mathbf{s}_{ip1}, \mathbf{s}_{ip2}, \Delta T_i, \Delta T_{ip1} $ and minimizes\+: $ \min \textrm{penaltyInterval}( [ax, ay, omegadot } ]^T ) \cdot weight $. ~\newline
{\itshape ax} is calculated using the difference quotient (twice) and the x position parts of all three poses ~\newline
{\itshape ay} is calculated using the difference quotient (twice) and the y position parts of all three poses ~\newline
{\itshape omegadot} is calculated using the difference quotient of the yaw angles followed by a normalization to \mbox{[}-\/pi, pi\mbox{]}. ~\newline
 {\itshape weight} can be set using set\+Information() ~\newline
{\itshape penalty\+Interval} denotes the penalty function, see penalty\+Bound\+To\+Interval() ~\newline
