\doxysubsection{详细描述}
This class converts the \mbox{\hyperlink{namespacecostmap__2d}{costmap\+\_\+2d}} into a set of lines (and points) 

The conversion is performed in three stages\+:
\begin{DoxyEnumerate}
\item Clusters in the costmap are collected using the DBSCAN Algorithm (\href{https://en.wikipedia.org/wiki/DBSCAN}{\texttt{ https\+://en.\+wikipedia.\+org/wiki/\+DBSCAN}}) Reference\+: Ester, Martin; Kriegel, Hans-\/\+Peter; Sander, Jörg; Xu, Xiaowei, A density-\/based algorithm for discovering clusters in large spatial databases with noise. Proceedings of the Second International Conference on Knowledge Discovery and Data Mining. AAAI Press. 1996. pp. 226–231. ISBN 1-\/57735-\/004-\/9.
\item In the subsequent stage, the convex hull of each cluster is determined using the monotone chain algorithm aka Andrew\textquotesingle{}s algorithm\+: C++ implementation example\+: \href{http://www.algorithmist.com/index.php/Monotone_Chain_Convex_Hull}{\texttt{ http\+://www.\+algorithmist.\+com/index.\+php/\+Monotone\+\_\+\+Chain\+\_\+\+Convex\+\_\+\+Hull}} ( GNU Free Documentation License 1.\+2 ) Reference\+: A. M. Andrew, \char`\"{}\+Another Efficient Algorithm for Convex Hulls in Two Dimensions\char`\"{}, Info. Proc. Letters 9, 216-\/219 (1979).
\item In the third step extract lines from each polygon (convex hull) if there exist at least a predefined number of support points.
\end{DoxyEnumerate}

The output is a container of polygons (but each polygon does only contain a single vertex (point) or two vertices (line)
