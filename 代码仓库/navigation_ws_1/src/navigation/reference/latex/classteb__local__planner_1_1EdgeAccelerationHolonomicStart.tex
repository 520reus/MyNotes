\doxysubsection{详细描述}
Edge defining the cost function for limiting the translational and rotational acceleration at the beginning of the trajectory. 

The edge depends on three vertices $ \mathbf{s}_i, \mathbf{s}_{ip1}, \Delta T_i $, an initial velocity defined by \mbox{\hyperlink{classteb__local__planner_1_1EdgeAccelerationHolonomicStart_a0e15a0dd3fba7f8f8bffe4bcc644910a}{set\+Initial\+Velocity()}} and minimizes\+: ~\newline
 $ \min \textrm{penaltyInterval}( [ax, ay, omegadot ]^T ) \cdot weight $. ~\newline
{\itshape ax} is calculated using the difference quotient (twice) and the x-\/position parts of the poses. ~\newline
{\itshape ay} is calculated using the difference quotient (twice) and the y-\/position parts of the poses. ~\newline
{\itshape omegadot} is calculated using the difference quotient of the yaw angles followed by a normalization to \mbox{[}-\/pi, pi\mbox{]}. ~\newline
{\itshape weight} can be set using set\+Information(). ~\newline
{\itshape penalty\+Interval} denotes the penalty function, see penalty\+Bound\+To\+Interval(). ~\newline
The dimension of the error / cost vector is 3\+: the first component represents the translational acceleration, the second one the strafing acceleration and the third one the rotational acceleration. \begin{DoxySeeAlso}{参见}
