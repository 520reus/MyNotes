\doxysubsection{详细描述}
Edge defining the cost function for keeping a minimum distance from obstacles. 

The edge depends on a single vertex $ \mathbf{s}_i $ and minimizes\+: ~\newline
 $ \min \textrm{penaltyBelow}( dist2point ) \cdot weight $. ~\newline
{\itshape dist2point} denotes the minimum distance to the point obstacle. ~\newline
{\itshape weight} can be set using set\+Information(). ~\newline
{\itshape penalty\+Below} denotes the penalty function, see penalty\+Bound\+From\+Below() ~\newline
