\doxysubsection{详细描述}
Edge defining the cost function for limiting the translational and rotational acceleration at the end of the trajectory. 

The edge depends on three vertices $ \mathbf{s}_i, \mathbf{s}_{ip1}, \Delta T_i $, an initial velocity defined by \mbox{\hyperlink{classteb__local__planner_1_1EdgeAccelerationGoal_a1cbc1c3d2bcada6779e25cee432db967}{set\+Goal\+Velocity()}} and minimizes\+: ~\newline
 $ \min \textrm{penaltyInterval}( [a, omegadot ]^T ) \cdot weight $. ~\newline
{\itshape a} is calculated using the difference quotient (twice) and the position parts of the poses ~\newline
{\itshape omegadot} is calculated using the difference quotient of the yaw angles followed by a normalization to \mbox{[}-\/pi, pi\mbox{]}. ~\newline
{\itshape weight} can be set using set\+Information() ~\newline
{\itshape penalty\+Interval} denotes the penalty function, see penalty\+Bound\+To\+Interval() ~\newline
The dimension of the error / cost vector is 2\+: the first component represents the translational acceleration and the second one the rotational acceleration. \begin{DoxySeeAlso}{参见}
