Create a graph and sample points in the global frame that can be used to explore new possible paths between start and goal. 

This version of the graph samples keypoints in a predefined area (config) in the current frame between start and goal. ~\newline
Afterwards all feasible paths between start and goal point are extracted using a Depth First Search. ~\newline
Use the sampling method for complex, non-\/point or huge obstacles. ~\newline
You may call \mbox{\hyperlink{classteb__local__planner_1_1ProbRoadmapGraph_a8d532f9cbcd43d39da0c28a5b1c7e6e6}{create\+Graph()}} instead.
