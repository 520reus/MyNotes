\doxysubsection{详细描述}
The H-\/signature in three dimensions (here\+: x-\/y-\/t) defines an equivalence relation based on homology using theorems from electro magnetism. 

The H-\/\+Signature depends on the obstacle configuration and can be utilized to check whether two trajectores belong to the same homology class. Refer to\+: ~\newline

\begin{DoxyItemize}
\item S. Bhattacharya et al.\+: Identification and Representation of Homotopy Classes of Trajectories for Search-\/based Path Planning in 3D, 2011 
\end{DoxyItemize}

Calculate the H-\/\+Signature of a path 

The implemented function accepts generic path descriptions that are restricted to the following structure\+: ~\newline
The path is composed of points T and is represented by a std\+::vector$<$ T $>$ or similar type (std\+::list, std\+::deque, ...). ~\newline
Provide a unary function with the following signature {\ttfamily  std\+::complex$<$ long double $>$ (const T\& point\+\_\+type) } that returns a complex value for the position (Re($\ast$)=x, Im($\ast$)=y).

T could also be a pointer type, if the passed function also accepts a const T$\ast$ point\+\_\+\+Type.

Check if two candidate classes are equivalent 

If the absolute value of the H-\/\+Signature is equal or greater than 1, a loop (in x-\/y) around the obstacle is indicated. Positive H-\/\+Signature\+: \mbox{\hyperlink{classteb__local__planner_1_1Obstacle}{Obstacle}} lies on the left hand side of the planned trajectory Negative H-\/\+Signature\+: \mbox{\hyperlink{classteb__local__planner_1_1Obstacle}{Obstacle}} lies on the right hand side of the planned trajectory H-\/\+Signature equals zero\+: \mbox{\hyperlink{classteb__local__planner_1_1Obstacle}{Obstacle}} lies in infinity, has no influence on trajectory
